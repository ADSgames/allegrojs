jAllegro examples

exhello.js!

Hello World! in jAllegro. This example initialises graphics and displays a centered 'Hello World!' message in the middle of display. It only takes 4 lines to do all that, so check it out!

exbmp.js

Bitmap loading and blitting. This example loads a bitmap from filea and displays it stretched on screen once it has loaded. It requires whooping six lines of code to make it happen.

exbounce.js

Motion, collision and sound. This example displays a ball bouncing accross the screen making a sound every timwe it bounces.

exgame.js

Example game. This example is a simple game of picking up apples from the ground. Each apple gives you a point.
